ОРГАНИЗАЦИЯ: Пример — Служба приёма обращений

ЦЕЛЬ (как работает мелкая фирма):
- Все жалобы копятся в ОДНОЙ таблице Excel.
- Раз в неделю (или по мере накопления) прогоняете файл через шаблон «Классификация жалоб».
- Помощник читает текст каждой строки и заполняет колонку «Отдел» по правилам ниже.
- Сохраняете файл и в Excel фильтруете/сортируете по «Отдел» — каждый ответственный видит свои строки.

КОЛОНКА С ТЕКСТОМ ЖАЛОБЫ:
- Ищи колонку: «Текст_жалобы», «Текст», «Жалоба», «Обращение», «Описание».
- Если неясно — самая длинная текстовая колонка.

НОВЫЕ КОЛОНКИ (добавить справа, в таком порядке):
1. Категория
2. Дополнительные_категории
3. Причина_выбора_категории
4. Отдел
5. Дополнительные_отделы
6. Приоритет
7. Краткое_резюме
8. Номер_обращения
9. Статус
10. Дата_регистрации
11. Дата_обработки

КОЛОНКИ Номер_обращения, Статус, Дата_регистрации, Дата_обработки:
- Оставь ПУСТЫМИ — их заполнит приложение при создании черновика.
- Статус по умолчанию: «Новая». Дата_обработки — когда сотрудник закроет обращение.

РАЗРЕШЁННЫЕ ОТДЕЛЫ (используй ТОЛЬКО эти названия в колонке «Отдел»):
- Служба поддержки
- Бухгалтерия
- Документооборот
- Контроль качества
- Руководитель смены
- IT-поддержка

НЕОДНОЗНАЧНЫЕ СЛОВА (читай ВЕСЬ текст строки, не одно слово):

«ОПЛАЧЕНО», «ОПЛАТА», «ВЫПЛАТА»:
- Деньги не пришли, неверная сумма, переплата, задержка → Ошибка в выплате → Бухгалтерия.
- Кнопка «оплачено», сайт, личный кабинет, статус не обновился → Технический сбой → IT.
- В тексте оба смысла или смысл неясен → Требует проверки → Руководитель смены.
- В Причина_выбора_категории обязательно напиши, как понял контекст (почему IT или бухгалтерия).

ЕСЛИ В ОДНОЙ ЖАЛОБЕ НЕСКОЛЬКО ПРОБЛЕМ:
1. Найди ВСЕ подходящие категории.
2. Категория — главная по срочности: выплата > грубость > документы > консультация > техсбой > задержка ответа.
3. Дополнительные_категории — остальные через «; » (точка с запятой и пробел).
4. Отдел — по главной категории.
5. Дополнительные_отделы — отделы по дополнительным категориям, через «; ».
6. Приоритет — самый высокий из найденных проблем.
7. Причина_выбора_категории — 1–2 коротких предложения: какие проблемы найдены и почему выбрана главная.
8. Не игнорируй второстепенные проблемы.

КАТЕГОРИИ И МАРШРУТИЗАЦИЯ:

1. Задержка ответа
   Признаки: не отвечают, долго жду, игнорируют, не перезвонили
   Отдел: Служба поддержки
   Приоритет: средний

2. Ошибка в выплате / начислении
   Признаки: не начислили, неправильная сумма, переплата, задержка выплаты
   Отдел: Бухгалтерия
   Приоритет: высокий

3. Проблема с документами
   Признаки: не приняли документы, потеряли, запросили повторно, справка
   Отдел: Документооборот
   Приоритет: средний

4. Грубое обращение
   Признаки: хамство, грубили, некорректно, оскорбили
   Отдел: Контроль качества
   Приоритет: высокий

5. Неправильная консультация
   Признаки: неверно сказали, дали не ту информацию, запутали
   Отдел: Руководитель смены
   Приоритет: средний

6. Технический сбой
   Признаки: сайт не работает, ошибка в системе, не загружается
   Отдел: IT-поддержка
   Приоритет: средний

ЕСЛИ КАТЕГОРИЯ НЕЯСНА:
- Категория: Требует проверки
- Дополнительные_категории: оставь пустым
- Отдел: Руководитель смены
- Дополнительные_отделы: оставь пустым
- Приоритет: низкий
- Причина_выбора_категории: кратко — почему не удалось однозначно классифицировать
- Краткое_резюме: 1 предложение по тексту. Факты не выдумывай.

ПРИОРИТЕТЫ: только «высокий», «средний», «низкий».

ФОРМАТ ОТВЕТА:
- Только валидный CSV. Текстовые поля с запятыми — в двойных кавычках.
